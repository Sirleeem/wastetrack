Bauchi local images for the WasteTrack landing page
===================================================

Place your own photos here (JPG or PNG). Keep these exact filenames
to replace what the site uses (no code change needed):

  suburb.jpg    - Hero background + resident cards
  college.jpg   - Collection team value prop
  lake.jpg      - Authority value prop + mission photo
  hospital.jpg  - Authority desk service + priorities
  gubi.jpg      - Field collection service + priorities
  hero.jpg      - Optional alternate hero (copy of suburb/lake if unused)

Tips:
  - Landscape photos work best (about 1600x900 or wider)
  - Prefer real Bauchi streets, markets, drains, collection trucks
  - File size under ~500 KB each is good for mobile

Current defaults are free Wikimedia Commons photos of Bauchi / Bauchi area.
