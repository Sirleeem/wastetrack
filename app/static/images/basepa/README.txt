BASEPA / Bauchi field photos for WasteTrack
==========================================

Folder: app/static/images/basepa/

These filenames are used on the landing page. Overwrite any file with a real
photo from the BASEPA Facebook page (or your own field shots) using the SAME name.

Required filenames
------------------
  hero.jpg              Full-width top hero background
  evacuation.jpg        Residents / overflow value card + priorities
  field-work.jpg        Collection officers card + drains priority
  collection-centre.jpg Authority desk service tile + value card
  community.jpg         Resident report service tile
  environment.jpg       Mission / about BASEPA section photo
  recycling-plant.jpg   Field collection service + delayed evacuation card

How to add real BASEPA photos (from Facebook online)
----------------------------------------------------
1. Open: https://www.facebook.com/p/Bauchi-State-Environmental-Protection-Agency-100069222459121/
2. Open a post with trucks, collection centres, or evacuation work
3. Save the photo to this folder
4. Rename it to one of the filenames above (replace the existing file)
5. Commit and push, or redeploy on Render

Tips
----
- Prefer landscape photos of trucks, tippers, collection centres, markets, desilting
- Keep each file under ~800 KB if possible (resize for web)
- No code change needed after replacing files

Note: Facebook blocks automatic downloads from this development environment,
so photos must be saved manually from the BASEPA page.
